Detailed Description

An implementation of PRR.

Introspection did not find any typical Config paths.

PRR is an algorithm that determines TCP's sending rate in fast recovery. PRR avoids excessive window reductions and aims for the actual congestion window size at the end of recovery to be as close as possible to the window determined by the congestion control algorithm. PRR also improves accuracy of the amount of data sent during loss recovery.


Attributes

  • ReductionBound: Type of Reduction Bound
    • Set with class: ns3::EnumValue

    • Initial value: SSRB
    • Flags: construct write read

No TraceSources are defined for this type.
Size of this type is 48 bytes (on a 64-bit architecture).
