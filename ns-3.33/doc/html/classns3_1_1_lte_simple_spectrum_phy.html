Detailed Description

The LteSimpleSpectrumPhy models the physical layer of LTE This class is used to test Frequency Reuse Algorithms, it allow to get SpectrumValue from channel and pass it to test script by trace mechanism.

When m_cellId is 0, all received signals will be traced, if m_cellId > 0, only signals from specified Cell will be traced.
