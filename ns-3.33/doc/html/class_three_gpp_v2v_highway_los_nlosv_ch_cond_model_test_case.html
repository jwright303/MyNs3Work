Detailed Description

Test case for the 3GPP V2V Highway channel condition models (probabilistic model for LOS/NLOSv states).

It determines the channel condition multiple times, estimates the LOS probability and compares it with the value given by the formulas in 3GPP TR 37.885, Table 6.2-1. Methodology from channel-condition-model- test-suite.cc is used.
