Detailed Description

Carries information about the LOS/NLOS channel state.

Introspection did not find any typical Config paths.

Additional information about the channel condition can be aggregated to instances of this class.


No Attributes are defined for this type.
No TraceSources are defined for this type.
Size of this type is 40 bytes (on a 64-bit architecture).
