Detailed Description

Test case for the classes ThreeGppV2vUrbanChannelConditionModel, and ThreeGppV2vHighwayChannelConditionModel to test their code to deterministically determine NLOS state.

The test checks if the channel condition is correctly determined when a building is deployed in the scenario. Methodology from buildings-channel-condition-model-test.cc is used.
