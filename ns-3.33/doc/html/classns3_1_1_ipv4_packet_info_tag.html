Detailed Description

This class implements Linux struct pktinfo in order to deliver ancillary information to the socket interface.

This is used with socket option such as IP_PKTINFO, IP_RECVTTL, IP_RECVTOS. See linux manpage ip(7).

See also SocketIpTosTag and SocketIpTtlTag

The Tag does not carry the Local address (as it is not currently used to force a source address).

This tag in the send direction is presently not enabled but we would accept a patch along those lines in the future.
